# **App Name**: Daily Sangama

## Core Features:

- Data Fetching: Fetch 'thought for the day', song, shloka, and panchanga data from Firestore.
- Category Display: Display data in four categories: thought for the day, song, shloka, and panchanga, each in its respective section.
- Audio Playback: Enable audio playback for songs with a provided media URL.
- Firestore Integration: Store and retrieve data, including text and media links, from Firestore database.

## Style Guidelines:

- Primary color: Deep saffron (#FF9933) to represent wisdom and spirituality.
- Background color: Very light yellow (#FAF9F6), providing a clean and serene backdrop.
- Accent color: Sky blue (#87CEEB) for a touch of divinity and peace.
- Headline font: 'Literata' serif for a classic, readable feel.
- Body font: 'Inter' sans-serif to pair with Literata; modern and clear.
- Use simple, clear icons to represent each category (thought, song, shloka, panchanga).
- Clean, card-based layout to present information clearly, suitable for daily content.
